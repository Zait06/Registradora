# Registradora
## Prerequisites
- Rust
- Tauri

### Install rust
Please, install rust following the [documentation Rust](https://www.rust-lang.org/tools/install)

### Install Tauri
Please, isntall tauri following the [documentation Tauri](https://tauri.app/v1/guides/getting-started/tauri-cli)

## Run project
### Install project
```pwsh
yarn install
```

### Run project
```pwsh
yarn tauri dev
```
